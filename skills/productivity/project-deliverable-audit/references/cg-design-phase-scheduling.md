## Root Cause

IFC depends on DD Approval (not submission). Complex MEP DD approval takes until 4-18 Aug → IFC needs +19-27 WD → misses deadline. Only Simple items (Stair approved 14 Jul, Terrace Shade approved 28 Jul) with 7 WD CG review can clear the full cycle.

## Recommended Response to CG

**3-phase approach:**

| Phase | End Date | What's Ready |
|-------|----------|-------------|
| Phase 1 ✅ | **21 Aug** | All DD + All Materials + Simple IFC |
| Phase 2 | **15 Sep** | Complex IFC (MEP, AV, FLS, Stramp) |
| Phase 3 | **30 Sep** | Exhibition IFC + Coordination — aligns with handover |

**Mitigations to request:**
1. Parallel IFC prep during DD review (saves 8-10 WD)
2. 7 WD CG review for medium-complexity IFC items
3. Split MEP IFC — Basement IFC early, upper floors later
4. 4-week extension for remaining IFC packages
