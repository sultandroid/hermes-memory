## ⚠️ DD vs IFC Correction (per DMP Rev C03)

The contract labels Stage 4 as "IFC Package" but the DMP Rev C03 defines a split that changes interpretation:

- **Stage 4-A DD (LOD 300-350)**: 251 drawings DELIVERED — this is what the register contains
- **Stage 4-B IFC (LOD 350-400)**: NOT STARTED — this is what the contract actually means by "IFC"

When writing reports: do NOT call the 251 drawings "IFC". They are **DD (Stage 4-A)**. The contract's terminology reflects an aspirational target that was later refined by the DMP. This distinction matters because the user monitors Stage 4-B as a separate, unbudgeted work package.
