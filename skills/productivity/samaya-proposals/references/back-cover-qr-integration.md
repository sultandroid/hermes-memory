# Back Cover QR Code Integration

When adding a QR code to the back cover of an HTML print-ready document (navy/dark background with gold accents), embed it within an existing contact column rather than as a standalone row.

## Design Pattern

## Rules

- **Place inside the "Online" column** of the 3-column contact grid (Factory / Phone / Online) вҖ” not as a separate row below. This keeps the layout compact and intentional.
- **Gold border frame**: `border:0.6px solid var(--gold); border-radius:1.5mm; padding:1.5mm;` wraps the QR image. Matches the gold accent (diamond separators, gradient lines) used elsewhere on the page.
- **QR size**: 20mm (small enough to fit without breaking the column width, large enough to scan). For standalone rows, use 28mm.
- **QR background**: The QR image itself should have a white embedded background (generated by API). The gold frame wraps it with 1.5mm breathing room.
- **URL text**: Already present above the QR in the column вҖ” no need for a separate URL label below the QR.

## QR Code Generation

```bash
# Generate QR for a URL, white background
curl -sL -o assets/img/brand/qr-profile.png \
  "https://api.qrserver.com/v1/create-qr-code/?size=300x300&data=https://samaya-factory.com&margin=10"
```
