## Interface Matrix Key (from SOW pages 70-72)

- **R** = Responsible (does the work)
- **A** = Accountable (signs off)
- **C** = Consulted
- **I** = Informed
- MoC = Ministry of Culture (Employer)
- MC = Main Contractor / Samaya

## Practical Use

When building delay/EOT claims, reference the specific SOW clause + matrix item:

> "Per SOW §2.2 (Item 2.12 in the Interface Responsibility Matrix), exhibition text is an MoC-supplied item. As of [date], this has not been received, impacting the graphic design critical path."

This is more effective than generic "client delay" language because it points to a specific contractual obligation.
