## Original Schedule from Moharram Bakhoum

Original engineering/design phase was 216 days. Key activities:

- Assessment & Survey: 51 days
- Technical Design 50%: ~70 days (all disciplines)
- Technical Design 90%: ~88 days
- Technical Design 100% IFC: ~27 days
- Fabrication Drawings: 11 days
- As-Built Drawings: 39 days
- Total design phase: 216 days

SOW/ER required compressing to 3 months.

## Key Schedule Logic Decisions

1. Prequalification must complete BEFORE 50% design — specialized trades need to be known
2. Material submittals run BETWEEN 50% approval and 90% completion
3. All disciplines run in PARALLEL, not sequential
4. 3-day review turnaround is the critical enabler for compression
5. Phased IFC release (Structural → MEP → Arch → Specialized) protects construction start
6. Longer durations for museum-specific items: Arch 50%=12-15d, Showcase=10-14d, BIM=10d
7. Graphic design starts with placeholder content — don't wait for client exhibition text
