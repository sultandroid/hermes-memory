Session reference: 2026-07-03 — CG comments from `Submission Comments + Arch.pdf` audited against SoW/ER/NRS SOW.

## Typical 5-Tier Verdict Examples

### ✅ Valid (in contract scope)

| CG Comment | Contract Source | Why |
|-----------|----------------|-----|
| Separate Rigging Register | SoW p.42 "Structural support for Exhibits, including rigging" | Rigging is a listed scope item |
| Dead load identification | SoW p.42 "structural loads, fixings and support" | Standard structural engineering |
| Work packages subdivided | ER p.6 "Compile IFC Documentation set, appropriately split into logical work packages" | ER explicitly requires package breakdown |
| Scenography drawings | SoW p.40 "Scenography and Models" as scope item; NRS SOW matrix row 44 "Scenography — Design & SD by Samaya" | Both documents confirm Samaya produces it |
| Furniture layouts | SoW p.15 "Joinery works and Loose Furniture"; NRS SOW matrix row 33 "FF&E — Design & SD by Samaya" | In scope |

### ✅ Valid by Code

| CG Comment | Code Reference | Why |
|-----------|---------------|-----|
| Universal Access Drawings | SoW p.13 "Universal Accessibility — Built Environment Guidelines for KSA 1431H-2010G"; SBC 201 | Mandatory building code |
| Access & Evacuation Plans | ER p.49-52 NFPA 101 / SBC 801 / BS 5266-1 | Fire code compliance |

### ⚠️ Borderline / Due Diligence

| CG Comment | Clause | Issue |
|-----------|--------|-------|
| Concrete core testing & scanning | SoW p.66 "Contractor shall conduct thorough investigations of all existing structures... before commencing work" | Due Diligence covers investigation but may not explicitly include destructive testing (core sampling). Negotiable — depends on interpretation of "thorough." |
| Geotechnical Report for external ramp | SoW p.42 mentions Stramp structural scope. SBC 303 cited as applicable code. | The ramp is in scope but geotechnical investigation is not explicitly listed as a deliverable. If required by SBC 303, it's code-driven. If not, it's a VO. |

### ❌ Invalid — No Contract Basis

| CG Comment | Why | SAMAYA Position |
|-----------|-----|----------------|
| Arch approval → Structural submission sequence | No contract clause prescribes discipline-level submission sequence. Contract defines scope milestones (Annex 4), not approval order. | New procedural gate. Schedule impact — notify under Art. 14. |
| Dates match approved schedule | The only approved program covers AV-only scope (86 days, ends 04/06/2026). Arch/Struct scope has no approved program. | Circular requirement — program must be revised first. |

## Quick Reference: Scope Items by Discipline

### Architecture (in scope for Samaya)
- Scenography drawings (SoW p.40, NRS matrix row 44)
- FF&E / Furniture layouts (SoW p.15, NRS matrix row 33)
- Universal Access / Accessibility (SBC 201, SoW p.13)
- Evacuation plans (NFPA 101, SBC 801, ER p.49-52)

### Structural (in scope for Samaya)
- Structural modifications (SoW §8.2, §3.10)
- Rigging design (SoW p.42, NRS matrix row 35)
- Dead load identification (standard engineering)
- Core testing (borderline — Due Diligence clause SoW p.66)
- Geotechnical investigation (borderline — SBC 303 cited, but not explicit deliverable)

### Not Samaya's Scope
- Scenography narrative (NRS is lead designer — but Samaya produces drawings per SoW p.40)
- Furniture design intent (NRS — but Samaya produces shop drawings per NRS matrix row 33)
