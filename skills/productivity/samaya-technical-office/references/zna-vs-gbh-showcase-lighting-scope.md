# ZNA Studio vs GBH: Showcase Lighting Scope Boundary

Found: 15-Jun-2026 during ZNA contract negotiation review.

## The Boundary

| Party | Role | Lighting Scope |
|-------|------|---------------|
| **ZNA Studio** (02_Lighting_Designer) | Specialist lighting designer | ALL exhibition/specialist lighting DESIGN — color temp, lux levels, beam angles, control integration, coordination with ambient |
| **GBH / Glasbau Hahn** (05_Showcases_Contractor) | Showcase fabricator | Integrated lighting HARDWARE within display cases — LED spots (SS1), linear LEDs (SL1), drivers, dimmers, light attic construction |
| **MEP Designer** (13_MEP_Designer) | Base-build MEP design | Base-build area lighting calculations only (§2.2: "exhibition lighting by 02_Lighting_Designer") |
| **MEP Contractor** (10_MEP_Contractor) | MEP installation | Installs base-build lighting per MEP Designer's design |

## Evidence

GBH profile shows they specify lighting components as part of case construction:
- `12x SS1 + 8m SL1; 3 driver, 3 dimmers` — hardware specs per case type
- Light attic engineering (structural integration of light panels)
- No mention of lux levels, color temperature, conservation light levels, or beam angle calculations

NRS comment on GBH submittal confirms design authority rests elsewhere:
> *"Can this row of spots be replaced by a linear light? See lighting designer's schedule"*

ZNA's email (15-Jun) explicitly states:
> *"LG-005 showcase lighting is not part of our scope, this is co-ordination only"*

## Gap

If ZNA does not provide showcase lighting design:
- GBH installs default LED spots without conservation-informed specifications
- No color temperature specification (risk: artifact damage from wrong spectrum)
- No lux level calculation per artifact type (risk: CG/conservation rejection)
- No coordination between showcase internal lighting and gallery ambient lighting
- No integration with lighting control system (dimming/scenes)

## Resolution Required

ZNA must either:
1. Include showcase lighting design in their base scope (as CG comment #1 requires), OR
2. Provide a clear specification data sheet that GBH can build to, even if ZNA calls it "coordination only"

The term "coordination only" is insufficient — GBH needs specific design parameters to fabricate.
