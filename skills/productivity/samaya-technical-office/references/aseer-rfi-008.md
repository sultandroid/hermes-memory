## Key Findings from May 28, 2026 Document Review

### Finding 1: Pull-and-slide for Types 6A/6B is ALREADY the NRS Design
The NRS original drawings (DT_4007, DT_4008, dated Jan 30, 2025) embedded in MOC-ASEER-SIC-1A0-TQ-0016.pdf **already show "PULL AND SLIDE"** for Types 6A and 6B. This is NOT a deviation from NRS intent.

### Finding 2: Type 1 Pull-and-Slide is BLOCKED
NRS explicitly rejected pull-and-slide for the recessed Type 1:
> *"We are also concerned that the pull-and-slide doors would require the showcase to be set forward of the wall cheeks to each side and this is not something that we are willing to accept as it changes the concept of the recessed showcase."*
**→ This requires a design decision. Options: (a) Keep recessed hinged door, (b) Modified recessed with reduced panel, (c) Accept design change.**

### Finding 3: Type 6B Back Panel Contradiction
- NRS drawing says: **"BACK IS GLASS"**
- NRS Feb 26 response says: **"The rear external panel should be powder coated metal"**
**→ NRS needs to clarify.**

### Finding 4: MoC RFI Parts B & C Are Blank
The submitted MOC RFI has no consultant or client sign-off. No MoC decision is on record.

### What the DMP Says About RFI-008 Items

**The brass and Corian substitutions are NOT just RFIs — they are CR Cat 2 (Change Requests):**

| CR Category | Description | Approval Path | RFI-008 Items |
|---|---|---|---|
| CR Cat 1 — Minor | Tech refinement, no cost/schedule impact, no design-intent change | Design Manager + PD sign-off | (none qualify) |
| CR Cat 2 — Major | Significant intent change OR Change Order to LS sum / programme impact | Full CG review + PMC + MoC formal Change Order under Contract Art. 14 | Brass · Corian · Type 1 door · Type 6B back panel |

**The TQ-0016 submitted was the wrong document type for material substitutions.** A TQ/RFI asks a question. A Change Request (CR) formally proposes a change requiring approval. Brass and Corian = CR Cat 2 → needs MoC Change Order.

### Correct Query Type Routing (DMP Sec 6.3.2)

| Query Type | Use When | Route To | SLA |
|---|---|---|---|
| RFI — Design | Design intent clarification | → NRS (DL/AoR) | 7 days |
| TQ — Engineering | Engineering / technical questions | → Samaya (EoR) | 7 days |
| Authority Query | Questions to MoC / authorities | → CG → MoC via Aconex | Per authority |
| CR Cat 1 | Minor change, no cost/intent impact | → DM + PD sign-off, logged on Aconex | — |
| CR Cat 2 | Design-intent change or cost/schedule impact | → CG + PMC + MoC formal Change Order | — |

**Current classification of RFI-008 items:**
- Type 1 recessed concept vs. pull-slide → **CR Cat 2** (design-intent change)
- Brass substitution (IGT powder coat) → **CR Cat 2** (employer-specified material change)
- Corian interior substitution → **CR Cat 2** (employer-specified material change)
- Type 6B back panel clarification → **RFI → NRS** (design clarification, not a change request yet)
- Motor assistance question → **TQ → Samaya/CG** (engineering question)
- Types 6A/6B pull-slide → **No action needed** — already NRS design intent
