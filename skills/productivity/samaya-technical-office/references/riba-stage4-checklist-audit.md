## PDF Extraction Pattern

```bash
# Convert RIBA checklist PDF to text
/opt/homebrew/bin/pdftotext /path/to/4-technical-design-checklist-pdf.pdf /tmp/riba4_checklist.txt

# List all numbered checklist items
grep -n "^[0-9]\.[0-9]" /tmp/riba4_checklist.txt

# Read the full extracted text
head -200 /tmp/riba4_checklist.txt
```

The PDF is an RTF-based locked Word form with checkboxes, notes fields, and completed-on date fields. Extraction via `pdftotext` produces clean plain text with section headers and item descriptions. The RTF instructions (Word 2003/2007/2010 unlock procedures) can be ignored — the text extraction works without them.
