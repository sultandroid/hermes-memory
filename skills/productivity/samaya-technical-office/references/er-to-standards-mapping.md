## When to Use This Mapping

- Replace all `ER-X.X` codes in a compliance matrix with the "Standard Ref" column
- The "Standard Name" is for reference only — the Arabic column header is **المعيار**
- Only use standards that are genuinely applicable — don't force-fit a standard to a non-standard requirement
- For requirements that have no applicable standard, use a generic descriptor (نطاق المشروع, تصميم BMA) or the internal section reference (ق.6, ق.10)
