# Schedule Compression — Museum Fit-Out (Design Phase)

## Context
Aseer Museum SOW/ER requires design completion in 3 months (90 days). The original Moharram Bakhoum schedule had 216 days for engineering alone. This reference documents how to compress it.

## Key DMP Gates
- D0: LOA / NTP
- D35: 50% Design
- D65: 90% Design
- D82: IFC
- D88: AFC
- D90: Site
- D300: TOC

## Phased Design Sequence (Compressed)

```
PREQUALIFICATION  (D0-D14)  → Subs/suppliers qualified before design
50% DESIGN        (D15-D33) → All disciplines parallel. Placeholders if client content missing.
MATERIAL SUBMITS  (D18-D49) → Between 50% and 90%. MANDATORY: must complete before 90% finalizes.
90% DESIGN + BIM  (D35-D61) → Uses approved materials from previous phase.
100% IFC          (D62-D90)  → Phased release: Structural → MEP → Arch → Specialized.
```

## Compression Methods
1. **Parallel disciplines** — ARC/STR/MEP all run simultaneously, not sequentially
2. **Fast-track approvals** — 3-day review turnaround (must be formally agreed with consultant)
3. **Overlapping stage gates** — 50%→90%→IFC have zero gaps, material submittals overlap with late 50%
4. **Continuous BIM** — model federation is ongoing, not restarted at each gate
5. **Specialized early start** — AV, lighting, showcase, scenography start alongside core disciplines
6. **Prequalification runs parallel** — with preliminaries and assessment, not sequential

## Critical Path Pitfalls
- **Material approvals MUST complete before 90%** — if they come after, 90% specs reference unapproved materials and need rework
- **Prequalification MUST complete before 50% specialized design** — AV, lighting, showcase designers need to know approved vendors
- **Client content (exhibition text, copyright, AV software) is the #1 risk** — these are MoC responsibilities per SOW §2.2
- **No buffer in 90-day schedule** — any client delay pushes the entire design timeline

## Realistic Baseline (if team/content not ready Day 1)
If team isn't staffed or client content isn't provided Day 1, the realistic design duration is ~117 days (4 months). Propose DMP revision:
- D35→D60, D65→D93, D82→D112, but keep D300 unchanged
- Phased IFC release protects the construction timeline even if design finishes late
