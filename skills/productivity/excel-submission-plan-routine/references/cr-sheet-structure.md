# CR Sheet Structure

## Ordering
1. **Architecture first** — all arch-related items from both CG sources
2. **Structure last** — all structural items

## CG Reference Naming
- "CG Comment N (Discipline) - DD Mon YYYY" for Mohammad Elbaz
- "Eng. Maged Zamzam email DD Mon YYYY" for Maged Zamzam
- Always use "Eng." prefix for all CG names

## Required Columns
| # | Header | Notes |
|---|--------|-------|
| 1 | CR | CR-01, CR-02, etc. Sequential |
| 2 | CG Comment | Full original CG comment text — do NOT truncate or rewrite |
| 3 | CG Reference | Source + date + reviewer |
| 4 | Response | Caveman style, Samaya perspective |
| 5 | Position | Short position statement |
| 6 | Status | Closed / Open / Noted / In Progress |
| 7 | Open / Closed | Same as Status but color-coded |
| 8 | Action Required | What needs to happen |
| 9 | Supporting Documents | File references |

## Color Coding
- Closed: Green fill `#E2EFDA`
- Open: Red fill `#FCE4EC`
- Noted: Blue fill `#D6E4F0`

## Summary Sheet Fields
Second sheet with: date generated, project, sources, total CRs, Architecture section (Closed/Open/In Progress counts), Structure section (Closed/Open/Noted counts), Next Steps list.
