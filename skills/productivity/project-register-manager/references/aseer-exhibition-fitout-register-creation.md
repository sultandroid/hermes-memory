## Key Lesson — Exact SOW Wording

Mohamed explicitly requested "list same as SOW Listed" — descriptions must use the exact wording from the SOW document, not paraphrased. For example:
- **Right:** "Setworks drawings" (from SOW 6.22)
- **Wrong:** "Detailed setwork drawings and shop drawings"
- **Right:** "Specific drawings and details for all Setworks and interior architecture related to the Exhibition Design" (from SOW 6.22)
- **Wrong:** "Setwork details and interior architecture coordination drawings"
