## Key Principles Applied

1. Contractor Specialist Package (left column) != Specialist (right column) - never mix the two types
2. Design and installation can be same entity (combined) or different entities (split)
3. Authority/statutory roles go to a separate section - not Samaya hires
4. Only add rows for roles Samaya is actually responsible to fill
5. All pending items noted as "to be submitted during project" per normal lifecycle
