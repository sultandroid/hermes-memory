## Key Lessons from This Session

- Long-lead items (Showcases 14wks, MEP 12-16wks) need expanded 50% packages
- SPEC.md is source of truth, register is derived output
- QC review (delegate to Kimi) before every delivery
- Appendix B PDF is authoritative for package list — not README
- Folders are numbered per Appendix B, not sequentially by creation order
- Draft email .md files are NEVER kept — delete immediately
