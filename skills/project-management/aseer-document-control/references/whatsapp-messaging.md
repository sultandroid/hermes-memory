# WhatsApp Messaging for Project Coordination

> Draft concise Arabic WhatsApp messages for Aseer Museum project coordination.

## Format

```
**م. [Name]** — السلام عليكم

بخصوص [topic/document ref]، [status update].

المطلوب من حضرتك:
1. [Action item 1]
2. [Action item 2]
3. [Action item 3]

وجزاك الله خير
```

## Rules

- **Arabic only** — user preference for WhatsApp
- **Direct, no fluff** — state the situation, then what's needed
- **Numbered action items** — makes it easy for the recipient to track
- **Reference document numbers** — always include the doc ref (e.g., MOC-MUS-ASE-1C0-1G-0001 Rev.01)
- **No AI language** — write like a colleague, not a consultant
- **End with** `وجزاك الله خير`

## Example: Structural DD Documents

```
**م. أحمد جاد** — السلام عليكم

بخصوص الـ Structural DD Documents (MOC-MUS-ASE-1C0-1G-0001 Rev.01)، وصلنا رد CG النهاردة بـ Code C تاني مرة. الـ 15 comment كلها لسه Open.

بالنسبة للـ core testing و rebar scanning — الشركات قدمت للاعتماد من الموقع، المطلوب من حضرتك تحديد نقاط الاختبار عشان نوجه الشركات تبدأ التنفيذ.

باقي البنود اللي محتاجين نشوفها:
- Geotechnical report — تحديد موقع الـ borehole
- Seismic importance factor (I=1.0) — دراسة كاملة حسب SBC 301
- Loading notation schedule — load pattern / case / combination
- Concrete cover + Masonry weights + Wind/Seismic refs — comparison tables مع أرقام SBC clauses

نحتاج نحدد المواعيد المستهدفة لكل بند عشان نعجل الـ Rev.02.

وجزاك الله خير
```
