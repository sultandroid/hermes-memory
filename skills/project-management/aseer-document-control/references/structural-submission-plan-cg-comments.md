## Key Contract-Doc Exclusions (from ASG SD Report)

The ASG SD Report (pre-contract, by Bluehaus/KM for MoC) explicitly excluded:

1. **Preparing analysis design models** of existing floors for new loads — "to be done by D&B contractor"
2. **Preparing design of new structural framing** required to stabilise new ID fitout elements
3. **Design of any structural additions/extensions** to existing buildings
4. **Design of any strengthening** required to existing floors/beams/columns
5. **No opening up works** were carried out to investigate existing building
6. **No loading plans** in the received as-built drawings set
7. **Missing as-built drawings**: ST-04A, ST-05, ST-06 sections, reinforcement details for basement retaining walls and core walls

Items 1-4 are now the contractor's responsibility. Items 5-7 are information gaps the contractor must fill.
