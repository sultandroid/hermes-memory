# Review & Approval Periods — Aseer Museum SMP

> Authoritative source for all review/approval turnaround times. Always cite the exact document clause. Never use a generic or assumed review period.

## Contractual Sources

| Review Type | Period | Source | Notes |
|---|---|---|---|
| **PMC Conformance Review** | **14 Calendar Days** | ER §2.4.A | Contractual minimum. Calendar days = all days including weekends. |
| **CG Technical Submittal Review** | **14 Working Days** | Comm Plan PL-0027 §7.1 | Working Days = Saturday–Thursday (Friday excluded). ≈ 17 Calendar Days. |
| **Design Instruction Request (DIR)** | 72 hours | ZD-0026 | Calendar hours. |
| **RFI / TQ Response** | 7 Working Days | Comm Plan PL-0027 | WD = Sat–Thu. |
| **Site Instruction Response** | 10 Working Days | Comm Plan PL-0027 | WD = Sat–Thu. |
| **Stage-Gate Review** | 14 Working Days | Comm Plan PL-0027 §9 | WD = Sat–Thu. |
| **MIR/WIR Sign-off** | 48 hours | DMP / Comm Plan | Calendar hours. |
| **ITP Review** | 7 days | DMP | Calendar days unless otherwise stated. |

## Conflict Rule

**Comm Plan §9.3:** When conflicting provisions exist across contract documents, the **most stringent** provision applies.

### Practical Implication

- PMC 14 CD (≈14 actual days) vs CG 14 WD (≈17 actual days): the **stricter of the two governs the minimum buffer**.
- When building submission schedules, use **14 CD** for PMC conformance review and **14 WD** (≈17 CD) for CG technical review — do NOT conflate the two.
- If a single deliverable requires both PMC and CG review, the combined buffer is **PMC 14 CD + CG 14 WD** (not overlapping, sequential per Comm Plan routing).

## Working Day Definition

Per Comm Plan §2.4: **Working Days = Saturday–Thursday**. Friday excluded.

## Sourcing Rules

1. **Only approved documents count** — Code B (Approved as Noted) or Code A (Approved). Code C documents are not approved and cannot be cited as authority.
2. **DMP Rev C04** (Code B) is the current approved DMP. Use it for design control procedures, review workflows, and ITP/TQ periods.
3. **Comm Plan Rev C02** (Code B) is the current approved Comm Plan. Use it for Stage-Gate SLAs, submittal routing, and review periods.
4. **ER §2.4.A** is the contractual authority for PMC review periods.
5. **ZD-0026** is the authority for DIR response times.
6. **Never source review periods from:** Code C documents, CG site instructions (unless formally incorporated), CG comments on unapproved submissions, or verbal agreements.

## Schedule Building Rule

When constructing a Deliverables Submission Schedule:
- Each row must cite its review period source (e.g., "14 CD per ER §2.4.A" or "14 WD per PL-0027 §7.1")
- DD dates are TBD until NRS confirms their production schedule
- IFC dates shown as Samaya Tech Office production (not NRS)
- Material Submittals shown as Project Team scope (not DD scope)
- No blanket "14 days" — always specify CD or WD with source
