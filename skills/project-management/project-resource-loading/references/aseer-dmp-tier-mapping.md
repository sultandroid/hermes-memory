## Common Alignment Errors

| Error | Symptom | Fix |
|-------|---------|-----|
| Wrong role name | T1 says "Construction Manager" not "Site Manager" | Rename all occurrences. Update card labels, table rows, RACI header (CM→SM), RBS abbrev list. |
| Extra T1 card | Tech Office Manager as separate T1 card | Move to T3. Add annotation if needed. |
| Internal roles in T2 | BIM Coords, Leads in T2 table | Move to T3. Replace with DMP's 14 design specialists. |
| Wrong firm assignment | AV Hardware assigned to NRS instead of Rawasen | Correct per DMP §5.1.1. AV #4 and Interactives #8 are both Rawasen (per KPR spelling). |
| Role removed instead of marked TBC | Mech. Interactives deleted from plan because Lumotion not approved | The ROLE is mandated by SOW §5.5 — keep it as TBC. Remove only the unapproved firm name. |
| Duplicate after rename | Two "SITE MANAGER" cards after renaming Construction Manager | Post-rename, audit §3.2 Construction chart specifically. Remove duplicate cards. |
| No reference chain | 3.2 Construction chart labeled "Per PD recommendation" | Replace with SOW §5.5 + §10 references. PD recommendations are not contract references. |
| Missing SOW §5.5 minimum role | Mech. & Electromech. Interactives absent from T2 | The SOW §5.5 "minimum staffing" list is a contract requirement, not optional. All roles in that list must appear in the plan. Cross-reference before finalizing. |
