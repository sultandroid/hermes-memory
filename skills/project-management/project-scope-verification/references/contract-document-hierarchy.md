# Contract Document Hierarchy (KSA Design-Build)

For MoC museum projects under Saudi Government Tenders Law (M/128), the document authority hierarchy is:

## Authority ranking (highest to lowest)

1. **The Contract** (main agreement) — payment terms, duration, governing law, scope boundaries
2. **Employer's Requirements (ER)** — technical scope, performance criteria, reference specifications
3. **Scope of Work (SoW)** — detailed description of work packages and exclusions
4. **BOQ / Pricing Schedule** — costed line items; Annex 4 for milestone-based lump-sum contracts
5. **Specifications** — CSI MasterFormat + project-specific discipline specs
6. **Plans & Procedures** — DMP, PEP, QMP, HSE Plan (execution methodology, not scope definition)

## Key features of Aseer-style lump-sum contracts

- No BoQ unit-rate re-pricing — quantity overruns sit with Contractor
- No advance payment
- Variations capped at +10%/-20% of contract value
- Annex 4 (Payment Schedule) = 15 deliverable bundles + final payments = defined scope
- ER clause: if ER conflicts with reference specs, ER overrides
