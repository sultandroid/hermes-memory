# Douglas Fir Timber Spec Reference

## Spec Requirements (061000 2.1.B.4)
- Scientific Name: Pseudotsuga Menziesii
- Source: Europe
- Density at 12% MC: 512 kg/m³
- Hardness: 3,160 N minimum
- MOE (Fir Dry): 1,357 kg/mm²

## Published Reference Values
Source: The Wood Database (https://www.wood-database.com/douglas-fir/)
- Average Dried Weight: 510 kg/m³ (32 lbs/ft³)
- Janka Hardness: 2,760 N (620 lbf)
- Elastic Modulus: 12.17 GPa (1,241 kg/mm²)
- MOR: 86.2 MPa (12,500 lbf/in²)
- Specific Gravity (12% MC): 0.51

## Gap
The spec values are higher than published averages for all three properties. These likely represent a specific structural grade (C24/C30 per BS EN 338) or European source per BS EN 942/EN 14081. A supplier TDS from a European timber supplier is needed to confirm compliance with the spec values. Mark as Partial with note: "Supplier datasheet to confirm grade-specific values."
