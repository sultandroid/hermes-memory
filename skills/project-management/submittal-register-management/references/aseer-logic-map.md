## Critical Path

Architecture delay at any stage cascades to ALL downstream registers:
- Arch 50% delay → MEP, FLS, AV, Showcase, Lighting start delay
- Arch 90% delay → Landscaping, Interactives start delay
- Arch IFC delay → entire project timeline shifts

Graphics and Model Maker are not on the critical path — they're client-data dependent.
