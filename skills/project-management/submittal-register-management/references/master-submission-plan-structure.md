# Master Submission Plan — Structure Reference

3-sheet xlsx generated by a Python script at `04_Registers/_scripts/Master_Submission_Plan.py`.

## Sheet 1 — Master Plan

Headers: Package, 50%/1st Sub, 90%, 100%, IFC/AFC, Depends On, Group, Review Buffer, Notes

Rows sorted by 50% date. Date-group headers ("Submit: 29 Jun") color-coded. Blue shading = parallel same-day submissions.

## Sheet 2 — Dependency Network

"If THIS is delayed → it blocks" table with lag estimates and critical path marking.

## Sheet 3 — Parallel Tracks

Chronological timeline showing who submits when across 5 parallel groups (0=Foundation, 1=Structural-feed, 2=Arch-feed, 3=Late, 4=Client-dep, 5=IFC).
