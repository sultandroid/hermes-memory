# Purchasing Info Pack — Door Hardware Worked Example

**Project:** Aseer Regional Museum
**Date:** June 2026
**Source specs:** NRS A2742-L20A (Doors) + A2742-P21 (Ironmongery) + A2742-1960/1961 (schedules)

## Trigger
User had NRS door hardware specs and datasheets. Asked to prepare a folder for the Purchasing Department to start finding suppliers.

## Steps Taken

### 1. Located Source Documents
Searched the subcontractor folder (`17_Doors_Metal_Contractors/`) for:
- NRS spec PDFs in `03_Specifications_and_Standards/11_Hardware_NRS_and_Datasheets/`
- Datasheets: ASSA ABLOY, Schlage L Series, Halspan, Moralt, Hilti
- Door schedule `A2742-1960.pdf` and ironmongery schedule `A2742-1961.pdf` in `02_Reference_Drawings/1950_Door_Types/`

### 2. Created Folder Structure
```
10_Purchasing/
  01_NRS_Specs/        — A2742-L20A.pdf, A2742-P21.pdf
  02_Datasheets/       — All manufacturer PDFs
  03_Drawings/         — A2742-1950/51/52 (door types), 1960 (schedule), 1961 (ironmongery)
  Door_Hardware_Purchasing_Info.docx  — styled summary
```

### 3. Key Findings from Spec Review
- **No brand specified** — NRS spec says "Single approved manufacturer or system supplier (to be confirmed)"
- **Finish:** Satin stainless steel Grade 316 (mandatory)
- **Standards:** EN, SBC, UL, NFPA 80/101
- **10 ironmongery sets** (A1 through K) covering ~39 doors across 4 levels
- Most common set: Type A2 (mortise lockset, 3x hinges, recessed handle)
- Fire-rated doors need closer + slide rail + fire certs

### 4. What Purchasing Was Asked To Do
- Contact 2-3 suppliers (ASSA ABLOY, Schlage, Dorma, HAFELE, Yale)
- Request Grade 316 satin stainless steel product ranges
- Get pricing + lead times
- Request fire certification docs (UL/SBC)
- Submit proposals to Tech Office for NRS approval

### 5. Style Guide Applied
The .docx was built using the Samaya Document Style Guide (navy headers, Calibri, alternating table rows, red note boxes, no emoji, no AI language).

## Output
- `17_Doors_Metal_Contractors/10_Purchasing/Door_Hardware_Purchasing_Info.docx` (44 KB, 8 sections, 11 tables)
- `_MANAGER_DASHBOARD/SITUATION_REPORT.md` updated to reference the new folder
