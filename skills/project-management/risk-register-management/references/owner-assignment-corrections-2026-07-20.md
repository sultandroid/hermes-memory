## Rule

- **Site/construction risks** → Construction Manager
- **Design/document risks** → Technical Office Manager
- FLS risks involving site execution (fire pump, enabling works) → Construction Manager
- Structural site investigation risks (testing, scanning, boreholes) → Construction Manager
- Submittal tracking, drawing numbering, CG submission sequence → Technical Office Manager

## Verification Protocol

When auditing risk register owners, check every risk whose mitigation involves:
- On-site execution (testing, installation, mobilization)
- Site-level coordination (fire pump, enabling works, NAFFCO)
- Structural investigation (core testing, rebar scanning, geotechnical)

If the primary mitigation is site-level, the owner is Construction Manager.
If the primary mitigation is document/design review, the owner is Technical Office Manager.
