## Audit Findings

| # | Issue | Severity | Detail |
|---|-------|----------|--------|
| 1 | Wrong contract value | 🟡 | States SAR 74,926,813.83 — actual is SAR 74,900,000 |
| 2 | Wrong completion date | 🔴 | States 01 March 2027 — contract handover is **30 Sep 2026** (5-month gap) |
| 3 | Wrong start date | 🟡 | States 01 May 2026 — NTP was 01 Dec 2025 |
| 4 | Wrong contract duration | 🟡 | States 10 months — correct duration but completion date contradicts it |
| 5 | Wrong FIDIC book | 🔴 | Written for FIDIC Red Book 2017 (Employer-designed). This is **Design & Build** (Yellow/Silver Book). Entire governance framework misaligned |
| 6 | Generic template | 🟡 | All 5,478 lines are generic. No mention of Aseer Museum, NRS, ZNA, Rawasin, Glasbau Hahn, Graphite, or any actual subcontractor |
| 7 | No reference to existing registers | 🟡 | Doesn't reference the project's actual subcontractor package register, prequalification register, or procurement register |
| 8 | No interface with existing specialist appointments | 🔴 | Doesn't address current status: 14 specialists in various stages of approval |
| 9 | No museum-grade requirements | 🟡 | No reference to Oddy testing, conservation standards, museum-grade finishes, or exhibition subcontractors |
| 10 | No linkage to current CG challenges | 🟡 | Doesn't address CG design challenges email, landscaping PQ rejection, or structural engineer approval delay |
| 11 | Unrealistic success criteria | 🟡 | 100% approved subs before mobilization, 0 LTI, 95% schedule adherence — aspirational, not grounded in reality |
| 12 | No exhibition subcontractor classification | 🟡 | Classification table (Critical/Major/Standard/Specialized) doesn't include exhibition-specific categories |

## Verdict

**❌ NOT READY FOR SUBMISSION** — CG will return Code C on dates alone (Mar 2027 vs Sep 2026). The FIDIC Red Book basis is fundamentally wrong for a Design & Build contract.

## What Must Be Fixed Before Resubmission

1. Correct dates: completion 30 Sep 2026, start 01 Dec 2025
2. Rewrite governance for Design & Build (Contractor-led design, not Employer-designed)
3. Add project-specific subcontractor register (packages 01-21 from repo)
4. Add exhibition specialist management section with Oddy/conservation requirements
5. Add current status of each package (approved/Code C/not appointed)
6. Add interface with CG approval process (PQ/MA/IFC flow)
7. Link to risk register (PRR-PRC-04, PRR-PRC-08, PRR-PRC-05)
