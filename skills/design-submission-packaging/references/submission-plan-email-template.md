# Submission Plan Email Template (CG/Consultant)

Reusable email template for communicating submission plan status to CG/consultant on Aseer Museum.

## Structure

1. Acknowledge the request/reminder
2. State what's attached now (Arch + Struct + AV plans)
3. State what's coming next (MEP, Lighting — with ETA and "coordinating with specialist" caveat)
4. List remaining disciplines with the phrase: "under preparation and coordination with the project team and specialists, and will be submitted progressively"
5. Reference the Master Submission Plan and deadline alignment
6. Apologize for any delay and reaffirm commitment to the agreed timeframe

## Discipline Grouping Rules

| Rule | Applies | Notes |
|------|---------|-------|
| MEP = Mechanical + Electrical + Plumbing | Always | Never list Electrical separately. Never add "(including Electrical)" after MEP — MEP inherently includes it. |
| Exclude: FFE, Model Maker, Oddy Testing | Always | Separate procurement/specialty tracks |
| Exclude: Graphics (if not yet approved), Showcase (if handled separately) | Confirm with user | These may be pending approvals or handled outside the design submission track |
| List separately: Acoustic, AV, CITC/Telecom, FLS, Interactives, Landscaping, Lighting, QA/Commissioning/Handover | When listing remaining disciplines | Confirm each with user before sending |

## Pitfalls

- **Reply All** — keep the full CC list (ACE, MOC, CG representatives) to maintain transparency on high-importance coordination emails.
- **Confirm ETA with user** before promising "tomorrow" — if not confident, use "within 2 days" or "by the end of the week" to avoid follow-ups.
- **Verify file paths** before sending — confirm the exact filenames exist in the register directory.
- **Cross-check with Master Submission Plan** — ensure individual discipline plan dates are synchronized with the master.
- **Never add "(including Electrical)" after MEP** — the user considers MEP to inherently include Electrical. Writing it out implies it doesn't.
- **Confirm Graphics and Showcase status** before including in the remaining disciplines list. Graphics may be "not yet approved" and Showcase may be handled separately.
- **Use "coordinating with specialist"** for MEP and Lighting when the specialist input is pending — this sets realistic expectations with CG.
- **Internal escalation** — when specialists are not yet appointed (Acoustic, CITC, Commissioning, Interactives, Landscaping) or site assessments are pending (FLS), send a separate internal email to the Project Manager explaining why these cannot be submitted to CG yet. The 3-month design window is a ticking clock.
