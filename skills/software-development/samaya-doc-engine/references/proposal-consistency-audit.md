# Proposal Consistency Audit — Exclusions vs BOQ

Before submitting a technical proposal, verify that **Section 34 (Assumptions, Exclusions & Clarifications)** is consistent with the **BOQ / Pricing Schedule**.

## What to Check

| Check | Why |
|-------|-----|
| Items listed as "Client responsibility" in exclusions | Must NOT have corresponding line items in the BOQ |
| Items listed as "Out of scope" in exclusions | Must NOT have priced line items |
| Items with price line items in BOQ | Must NOT be listed as excluded — unless it's a provisional sum or optional scope clearly marked |
| "By others" items | Verify those "others" exist and are assigned — if not, scope gap |

## Common Pitfall: Content Production

The most frequent contradiction found in RCRC proposal reviews:

- **Proposal Section 34 (Exclusions), X2**: "إنتاج المحتوى الإبداعي (Content Production) — الأفلام الوثائقية، المحتوى التفاعلي، التصميم الجرافيكي الإبداعي — هذا من اختصاص العميل أو جهة أخرى يعينها"
- **BOQ**: Contains priced line items for documentary films, interactive content, or creative graphic design
- **Problem**: The proposal tells the client this is their responsibility while simultaneously pricing it → contradiction that causes confusion during evaluation

## Fix Options

| Option | When to Use |
|--------|-------------|
| **Remove the exclusion** | If Samaya is pricing these items, they're in scope. Delete X2 from Section 34. |
| **Keep exclusion + add caveat** | If Samaya prices it but wants the client to supply core content assets (e.g., archival footage, historical photos). Add: "Content Production is included — client to supply raw assets and final approvals." |
| **Add provisional sum note** | If the price is a placeholder. Add: "Provisional — subject to client content brief." |

## Procedure

1. Extract all exclusion items from Section 34 (usually X1-X8 or similar)
2. Extract all BOQ line item descriptions
3. Map each exclusion to BOQ items — flag any that appear in both
4. Report contradictions to user before submission
5. After fixing, re-check that Section 34 matches the final BOQ
