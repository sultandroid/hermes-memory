## Critical Path Summary

The schedule shows **negative float on most design activities** from day one — the program is already compressed:

1. **Worst offenders**: Showcase Design (-34d at both 50% and 90%), BIM 90% (-34d), IFC 100% BIM (-35d), Architectural 50% (-32d)
2. **Zero-float items**: Plumbing Assessment (A1235), BIM Existing Model (A1237), Architectural BIM 50% (A1610, A1630)
3. **Convergent bottleneck**: All ~80 material submittals approved in same 5-day window → PO on 15-Mar-26
4. **Procurement gap**: PO (15-Mar) → Delivery (13-May) → Construction (19-Aug) — 3-month lead time for museum-specific items (showcases, AV, interactives, scenography)

## Key Dependencies

```
Site Surveys → Assessments (ARC, STR, MEP) → Approvals → 50% Design
50% Design → Approvals → 90% Design → BIM 90% + Clash Detection
90% BIM → IFC 100% → Approval → Fabrication Drawings
Fabrication Drawings → Material Submittals → PO → Delivery
Delivery + As-Builts → Construction → T&C → Handover
```
